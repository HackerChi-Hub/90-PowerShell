# 🧰 PowerShell 多功能脚本集合

> 🚀 一套面向 Windows 的 PowerShell 多功能脚本集合，覆盖 🖥️ 系统管理、🛡️ 安全、⚡ 优化、📈 监控等常见场景。

---

## 📌 项目简介

本项目整理了一组 PowerShell 脚本能力集合，聚焦 **系统信息采集、🌐 网络诊断与探测、🔒 安全检查、📊 性能监控、🤖 自动化运维** 等用途，适合用于日常维护、🧯 排障、✅ 基线核查与 🗂️ 批量管理。

## 🧩 功能总览（90+）

包含但不限于以下能力 👇

### 🖥️ 系统与硬件
- 🧾 系统信息
- 🧬 硬件详细信息
- 📦 软件安装列表
- 🔄 系统更新检查
- 🚀 系统启动项管理
- 🪟 打开系统功能（快捷入口类）

### 🌐 网络与连接
- 🛰️ 网络探测
- 🧪 端口扫描
- 🔌 网络连接监控
- 🧠 DNS 缓存管理
- 🧷 ARP 表查看

### 📁 文件与磁盘
- ⬇️ 文件下载
- ✂️ 文件处理
- 🧺 批量文件操作
- 🧮 文件哈希计算（用于校验/取证）
- 🔐 文件权限检查
- 💽 磁盘空间分析
- 🕵️ 可疑文件检测

### ⚙️ 进程 / 服务 / 任务
- 🏃 进程管理
- 🧩 服务管理
- 🗓️ 计划任务管理

### 🛡️ 安全与审计
- 📶 WiFi 密码显示
- 🧱 防火墙规则查看
- 🧷 注册表查询
- 📜 系统日志查看
- 👤 用户账户信息
- ✅ 系统安全扫描
- 🦠 恶意软件检测

### 📈 性能与监控
- 📊 系统性能监控

### 🧪 环境与配置
- 🌱 环境变量管理
